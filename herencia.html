Introducción a la Herencia en CSS

La herencia en CSS se refiere a la forma en que las propiedades CSS de un elemento padre se transmiten a sus elementos hijos. No todas las propiedades se heredan, pero muchas de las propiedades que afectan el texto y el diseño se heredan automáticamente.

Propiedades Heredables

Propiedades como color, font-family, font-size, line-height, text-align, y visibility se heredan por defecto.

Propiedades No Heredables

Propiedades como margin, padding, border, background, y width no se heredan por defecto. Estas propiedades deben ser definidas explícitamente para cada elemento.

Uso de inherit

Puedes forzar la herencia de cualquier propiedad utilizando la palabra clave inherit.
