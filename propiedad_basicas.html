Color y Fondo

color: Define el color del texto.

background-color: Establece el color de fondo de un elemento.
background-image: Aplica una imagen de fondo a un elemento.

Fuente y Texto

font-family: Define la fuente del texto.

font-size: Establece el tamaño del texto.

font-weight: Determina el grosor del texto.

text-align: Alinea el texto (izquierda, derecha, centro, justificar).

text-decoration: Añade decoraciones al texto (subrayado, línea a través, etc.).

Espaciado y Tamaño

margin: Establece el espacio exterior alrededor de un elemento.
padding: Define el espacio interior dentro de un elemento.
width: Ajusta el ancho de un elemento.
height: Ajusta la altura de un elemento.

Bordes

border: Establece el borde de un elemento (ancho, tipo y color).
border-radius: Aplica esquinas redondeadas a un borde.

Posicionamiento

position: static;
position: relative; top, right, bottom, left
position: absolute; top, right, bottom, left
position: fixed; top, right, bottom, left
position: sticky; top, right, bottom, left

Visualización y Flexbox

display: block, inline, flex, grid, none
flex: Ajusta la distribución de los elementos en un contenedor flexible.
justify-content: Alinea los elementos flexibles en el eje principal.
align-items: Alinea los elementos flexibles en el eje cruzado.

Modelo de Caja

box-sizing: Define cómo se calculan los valores de width y height (content-box, border-box).
opacity: Define la opacidad de un elemento.
z-index: Controla la superposición de elementos en capas.
