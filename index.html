Propiedades básicas de CSS

Color y Fondo
color: Define el color del texto.
background-color: Establece el color de fondo de un elemento.
background-image: Aplica una imagen de fondo a un elemento.
Fuente y Texto
font-family: Define la fuente del texto.
font-size: Establece el tamaño del texto.
font-weight: Determina el grosor del texto.
text-align: Alinea el texto (izquierda, derecha, centro, justificar).
text-decoration: Añade decoraciones al texto (subrayado, línea a través, etc.).
Espaciado y Tamaño
margin: Establece el espacio exterior alrededor de un elemento.
padding: Define el espacio interior dentro de un elemento.
width: Ajusta el ancho de un elemento.
height: Ajusta la altura de un elemento.
Bordes
border: Establece el borde de un elemento (ancho, tipo y color).
border-radius: Aplica esquinas redondeadas a un borde.
Posicionamiento
position: Controla el método de posicionamiento de un elemento (static, relative, absolute, fixed, sticky).
top, right, bottom, left: Posiciona un elemento desde los bordes del contenedor.
Visualización y Flexbox
display: Define el modelo de caja de un elemento (block, inline, flex, grid, none).
flex: Ajusta la distribución de los elementos en un contenedor flexible.
justify-content: Alinea los elementos flexibles en el eje principal.
align-items: Alinea los elementos flexibles en el eje cruzado.
Modelo de Caja box-sizing: Define cómo se calculan los valores de width y height (content-box, border-box).
opacity: Define la opacidad de un elemento.
z-index: Controla la superposición de elementos en capas.

Herencia

La herencia en CSS se refiere a la forma en que las propiedades CSS de un elemento padre se transmiten a sus elementos hijos. No todas las propiedades se heredan, pero muchas de las propiedades que afectan el texto y el diseño se heredan automáticamente. Aquí tienes algunos puntos clave:
Propiedades Heredables: Propiedades como color, font-family, font-size, line-height, text-align, y visibility se heredan por defecto.
Propiedades No Heredables: Propiedades como margin, padding, border, background, y width no se heredan por defecto. Estas propiedades deben ser definidas explícitamente para cada elemento.
Uso de inherit: Puedes forzar la herencia de cualquier propiedad utilizando la palabra clave inherit.

Flexbox

Ejes Principal y Secundario: Flexbox organiza los elementos en un contenedor en dos ejes: el eje principal (por defecto, horizontal) y el eje secundario (por defecto, vertical). Los elementos flexibles pueden alinearse y distribuirse a lo largo de estos ejes.

Contenedores Flex y Elementos Flex: Un contenedor flex se define aplicando display: flex; a un contenedor. Los hijos de este contenedor se convierten automáticamente en elementos flexibles y se organizan según las reglas de flexbox.
Dirección del Eje (flex-direction): Puedes cambiar la dirección en que los elementos se organizan utilizando la propiedad flex-direction. Las opciones incluyen row (por defecto), row-reverse, column, y column-reverse.
Alineación y Espaciado:

justify-content: Alinea los elementos a lo largo del eje principal. Opciones incluyen flex-start, flex-end, center, space-between, space-around y space-evenly.
align-items: Alinea los elementos a lo largo del eje secundario. Opciones incluyen stretch (por defecto), flex-start, flex-end, center y baseline.
align-self: Permite alinear un elemento individualmente, ignorando la configuración de align-items del contenedor.
Flexibilidad de los Elementos:

flex-grow: Controla cómo un elemento puede crecer en relación a los demás elementos flexibles.
flex-shrink: Define cómo un elemento puede encogerse en relación a los demás elementos flexibles.
flex-basis: Especifica el tamaño inicial de un elemento flexible antes de que se distribuya el espacio disponible.
Orden de los Elementos: Utilizando la propiedad order, puedes cambiar el orden visual de los elementos en un contenedor flex sin alterar el HTML.
Flexbox es especialmente útil para diseños unidimensionales donde necesitas que los elementos se distribuyan de forma equitativa dentro de un contenedor, sea horizontal o verticalmente. Facilita la creación de diseños responsivos y ajustables sin necesidad de flotaciones o posiciones complicadas.

Position con CSS

static:

Es el valor por defecto.
Los elementos se posicionan en el flujo normal del documento.
Las propiedades top, right, bottom y left no afectan a los elementos con position: static;.
relative:

El elemento se posiciona relativo a su posición original en el flujo del documento.
Las propiedades top, right, bottom y left se utilizan para ajustar su posición desde la original.
No saca al elemento del flujo normal, lo que puede afectar a otros elementos.
absolute:

El elemento se posiciona relativo a su contenedor más cercano que no tenga position: static; (puede ser un ancestro).
Se elimina el elemento del flujo normal del documento, por lo que no afecta a la posición de otros elementos.
Las propiedades top, right, bottom y left se utilizan para ajustar su posición.
fixed:

El elemento se posiciona relativo a la ventana del navegador (viewport).
Se elimina del flujo normal del documento.
Permanece fijo en la misma posición incluso cuando se hace scroll en la página.
Las propiedades top, right, bottom y left se utilizan para ajustar su posición.
sticky:

El elemento se comporta como relative hasta que alcanza un umbral definido por las propiedades top, right, bottom o left, en cuyo punto se comporta como fixed.
Es útil para elementos que deben permanecer visibles mientras se hace scroll hasta que se alcanza cierto punto en la página.

Estilos Responsivos

Los estilos responsivos permiten que un sitio web adapte su diseño y contenido a diferentes tamaños de pantalla y resoluciones, mejorando la experiencia del usuario en dispositivos móviles, tabletas y pantallas de escritorio.
Media Queries:

Las media queries permiten aplicar estilos CSS específicos según las características del dispositivo, como el tamaño de la pantalla.
Porcentajes y Unidades Relativas:

En lugar de usar píxeles fijos, se pueden usar porcentajes, em, rem y otras unidades relativas para que los elementos se ajusten proporcionalmente al tamaño de la pantalla.
Flexbox y CSS Grid:

Estas herramientas permiten crear diseños flexibles y adaptativos que pueden reorganizarse automáticamente según el tamaño de la pantalla.
Flexbox es ideal para diseños unidimensionales, mientras que CSS Grid es mejor para diseños bidimensionales.
Imágenes y Medios Responsivos:

Utilizar imágenes que se ajusten automáticamente al contenedor padre usando max-width: 100%; y height: auto;.
También se pueden usar diferentes fuentes de imágenes según el dispositivo con el elemento y los atributos srcset y sizes en .
Tipografía Responsiva:

Ajustar el tamaño de las fuentes de manera proporcional al tamaño del viewport utilizando unidades como vw (viewport width) y vh (viewport height).
Beneficios de los Estilos Responsivos

Mejor experiencia de usuario: Los sitios web se adaptan a diferentes dispositivos, proporcionando una navegación más cómoda y accesible.
Mejor SEO: Los motores de búsqueda favorecen los sitios web que son móviles y responsivos.
Mantenimiento simplificado: Un solo diseño que se adapta a todos los dispositivos es más fácil de mantener que múltiples versiones del mismo sitio.
